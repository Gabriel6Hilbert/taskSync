# Gestão de Tarefas - TaskSync

Sistema de **gestão de tarefas** desenvolvido com **Java**, utilizando **Spring Boot** e **Maven**, com integração ao **Google Calendar**. A solução permite o controle completo de tarefas com priorização, alertas e controle de status.

## 📁 Estrutura do Projeto

- `src/main/java` – Código-fonte da aplicação
- `src/main/resources` – Arquivos de configuração e templates
- `pom.xml` – Gerenciamento de dependências (Maven)
- `.gitignore`, `.gitattributes` – Arquivos de controle de versão

## 🚀 Tecnologias Utilizadas

- Java 17+
- Spring Boot
- Maven
- JPA/Hibernate
- MySQL
- Bootstrap 5
- JavaScript
- Google Calendar API

## ⚙️ Pré-requisitos

- JDK 17+
- Maven
- Docker e Docker Compose
- MySQL configurado via container
- (Opcional) Ferramenta CI/CD para automação do pipeline

## ✅ Funcionalidades

- [x] CRUD de tarefas
- [x] Definição de prioridades e status
- [x] Alertas por data e hora
- [x] Integração com Google Calendar
- [x] Interface responsiva (Bootstrap)
